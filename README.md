
# Medusa Brute-Force Lab – Projeto Completo

Este repositório contém um laboratório completo para simulação de ataques de força bruta usando **Medusa** no Kali Linux, 
com alvos Metasploitable 2 e DVWA.

## Estrutura
- README.md
- users.txt
- passwords.txt
- scripts/
- images/ (prints simulados)
- outputs/

## Conteúdos
Inclui wordlists, scripts, prints simulados e arquivos de saída para facilitar documentação do desafio DIO.
